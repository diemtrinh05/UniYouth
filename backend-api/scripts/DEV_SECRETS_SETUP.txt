UniYouth.Api secret setup (dev)
================================

1. Copy:
   scripts\dev-runtime-env.example.ps1

   to:
   secrets\dev-runtime-env.ps1

2. Điền giá trị thật cho các biến:
   - Jwt__Key
   - Email__Smtp__Password
   - PushNotifications__Fcm__ServiceAccountJsonPath
   - các biến Gmail/APNS nếu có dùng

3. Chạy stack như bình thường:
   - powershell -ExecutionPolicy Bypass -File .\scripts\face_stack.ps1 start
   - hoặc powershell -ExecutionPolicy Bypass -File .\scripts\public_stack.ps1 start

4. Script face_stack sẽ tự nạp:
   secrets\dev-runtime-env.ps1

5. Không commit file trong thư mục:
   secrets\
